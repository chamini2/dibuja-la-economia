Project for Prodavinci Article about economics of Venezuela http://factor.prodavinci.com/dibujalaeconomia/index.html

# Development

```
$ node --version
v12.22.12

$ npm --version
6.14.16
```

Install dependencies

```
npm install
```

And start the server

```
gulp serve
```
